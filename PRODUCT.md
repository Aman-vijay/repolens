# Product

## Register

product

## Users

Software engineers, technical leads, and product managers who are working with unfamiliar codebases. They need to quickly onboard, map out dependencies, search semantic chunks, chat with the codebase, and write structured plans for complex refactoring or feature integration.

## Product Purpose

RepoLens provides a clear, high-fidelity lens into any repository by providing automated structure extraction, semantic search over codebase chunks, and AI-assisted understanding. It helps developers move from feeling lost in code to having an execution-ready implementation plan.

## Brand Personality

- **Precise**: Code analysis requires absolute correctness. We prefer high-density, clean UI structure over fluffy visuals.
- **Supportive**: Spelunking through unfamiliar code is stressful. The tool should feel responsive, fast, and structured, guiding the user step-by-step.
- **Developer-Centric**: We speak in terms developers understand: file trees, language breakdowns, functions, imports, dependency paths. No overhyped AI buzzwords.

## Anti-references

- **AI-Slop Clichés**: Excessive purple/pink neon glow gradients, glowing particles, glassmorphism, floating chatbots that block content.
- **Nested Card Clutter**: Cards inside cards inside cards with borders and drop-shadows. Instead, use clear structural borders, flat grids, or unified layouts.
- **Vague Copy**: Promising "magical AI that writes all your code for you." RepoLens is focused on planning and understanding, not code generation.
